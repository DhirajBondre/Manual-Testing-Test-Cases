This project contains a detailed Manual Testing Test Case document created for the MakeMyTrip Flight Booking Module (One Way Flight). The purpose of this project is to validate the functionality, usability, and correctness of the flight booking process by executing different test scenarios.

# The following flight booking functionalities were tested:

# 1. Flight Menu Navigation
Verify Flight option is available on the homepage.
Validate Flight menu is visible as the first option.
Check menu highlight behavior when selected.
# 2. Trip Type Selection
Validate available trip types:
One Way
Round Trip
Multi City
# 3. Departure City Selection
Verify user can search departure city.
Validate suggestion dropdown functionality.
Check city selection behavior.
# 4. Destination City Selection
Verify user can search destination city.
Validate auto-suggestion list.
Confirm correct city selection.
# 5. Date Picker Validation
Verify calendar opens correctly.
Ensure only today’s date and future dates are selectable.
Validate past dates are disabled.
# 6. Passenger Selection
Validate Adults count
Maximum: 9
Default: 1
Validate Children count
Maximum: 6
Default: 0
Validate Infants count
Maximum: 6
Default: 0
# 7. Travel Class Selection
Verify available class options:
Economy Class (EC)
Premium Economy (PE)
Business Class
First Class
# 8. Flight Search Functionality
Validate Search button action.
Verify system displays available flights after valid input.
Confirm proper navigation to result page.
